# Roadmap

## Готово: 0.3

- Read-only аудит файлов, процессов и механизмов автозапуска.
- Локальная перепроверка находок через Kaspersky без удаления файлов.
- HTML, CSV и JSON отчеты.
- Безопасная история: неполные проверки не меняют baseline, разные области аудита сравниваются раздельно.
- Анализ payload в launcher-командах и пользовательских каталогов `AppData`/`LocalAppData`.
- Pester-тесты для ключевых сценариев корректности.

## Далее: 0.4

- Белый список доверенных файлов с SHA-256 и комментарием пользователя.
- Настройка чувствительности для уменьшения ожидаемых ложных срабатываний.
- Русификация HTML-отчета и более краткий экран сводки.
- Ротация старых отчетов и baseline с настраиваемым сроком хранения.

## Позже

- Карантин только после ручного подтверждения, с журналом и восстановлением.
- Проверка хэшей через внешний сервис только после явного включения.
- Подписывание релизных файлов цифровой подписью при наличии сертификата.
